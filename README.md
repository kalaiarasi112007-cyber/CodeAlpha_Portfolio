# 🌐 Kalai Portfolio Website

This is my personal portfolio website developed using HTML, CSS, and JavaScript. It showcases my skills, projects, and resume with a clean and responsive design.

---

## 🚀 Features

- 👤 About Me section  
- 🛠 Skills showcase  
- 💻 Projects display  
- 📄 Resume download & view  
- 📱 Responsive design  
- 🔄 Section-based navigation (no scrolling)  

---

## 💻 Projects Included

### 🧮 Calculator  
A simple calculator that performs basic arithmetic operations.

### 🖼 Image Gallery  
A responsive image gallery with grid layout and hover effects.

### ⏱ Task Time Tracker  
Tracks the time taken to complete a typing task using JavaScript timer functions.
